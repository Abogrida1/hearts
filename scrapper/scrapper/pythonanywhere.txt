# PythonAnywhere Deployment Instructions

## خطوات النشر على PythonAnywhere:

### 1. إنشاء حساب
- اذهب إلى https://www.pythonanywhere.com/
- أنشئ حساب مجاني جديد

### 2. رفع الملفات
- اذهب إلى "Files" tab
- ارفع جميع ملفات المشروع إلى مجلد جديد

### 3. إنشاء Web App
- اذهب إلى "Web" tab
- اضغط "Add a new web app"
- اختر "Flask" و Python 3.11

### 4. تثبيت المتطلبات
- اذهب إلى "Consoles" tab
- افتح Bash console
- نفذ الأوامر التالية:

```bash
cd your-project-folder
pip install -r requirements.txt
```

### 5. إعداد Web App
- في "Web" tab، اضغط على "Code" section
- Source code: اختر مجلد المشروع
- Working directory: اتركه فارغاً
- WSGI configuration file: عدل الملف ليصبح:

```python
import sys
path = '/home/yourusername/your-project-folder'
if path not in sys.path:
    sys.path.append(path)

from app import create_app
application = create_app()

if __name__ == "__main__":
    application.run()
```

### 6. إعداد Environment Variables
- في "Web" tab، اذهب إلى "Environment variables"
- أضف:
  - DEBUG=False
  - SECRET_KEY=your-secret-key-here

### 7. تشغيل التطبيق
- اضغط "Reload" في "Web" tab
- انتظر حتى يظهر "Running"
- اذهب إلى رابط التطبيق

## ملاحظات مهمة:
- لا يمكن التحميل المباشر على PythonAnywhere
- التطبيق يعرض معلومات الفيديو فقط
- استخدم الأدوات الخارجية للتحميل الفعلي

## استكشاف الأخطاء:
- تحقق من "Error log" في "Web" tab
- تأكد من تثبيت جميع المتطلبات
- تحقق من صحة WSGI configuration
